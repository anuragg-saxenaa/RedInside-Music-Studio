Artwork & Create — Which approach?

How should artwork generation and song metadata be structured in the DAW?

A

Track Art in Right Panel + Album Studio tab

Per-track artwork lives in the Right Panel alongside tags/notes (already song-context). The "Create" tab becomes "Album Studio" — generate album cover, set album title, artist name, track order. Apple Music-style.

✓ Artwork always in song context ✓ Album gets dedicated space △ Right panel gets busier
B

Artwork tab with Track / Album sub-tabs

Rename "Create" → "Artwork". Two sub-tabs: Track Art (generates for selected song, shows its artwork, sets per-song metadata) and Album Art (select multiple songs, generate album cover, set album title + artist). Single focused destination.

✓ Clean separation ✓ Doesn't crowd Right Panel ✓ Album workflow natural △ Video/Voice need a new home
C

Song Edit Panel (inline) + Create for album only

Each track row gets an "Edit" expanded panel (click ✎ icon) showing: artwork thumbnail, generate art button, per-song metadata fields (title, artist, BPM, key). Create tab handles album-level only: album cover, tracklist order, credits.

✓ Song editing in-place △ Inline panels = complex layout △ More implementation effort
